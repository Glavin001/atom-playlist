# Playlist

An [Atom editor]() plugin for queueing and playing music.

## Features

- [ ] Queue music from Sound cloud
- [ ] Play music from Sound cloud

## Keymaps

mapping | command
--- | ---
`cmd-alt-l` | Toggles the Playlist
